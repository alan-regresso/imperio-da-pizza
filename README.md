# imperio-da-pizza

O Império da pizza é um sistema web básico focado em facilitar o
recebimento de pedidos de uma pizzaria. A ideia é substituir o
caderninho de anotações e o telefone por um site simples, onde os
pedidos chegam de forma organizada e fácil de ler, evitando erros na
hora de anotar o sabor da pizza ou o endereço do cliente.
